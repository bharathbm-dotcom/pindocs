# pindocs

This repository hosts the API documentation website for the PIN APIs.

The documentation is generated from an OpenAPI specification (`data.yaml`)
and rendered as a web interface using Swagger UI.

## Files

- `wbs.html` – API documentation website
- `data.yaml` – OpenAPI specification

## Live Documentation

After enabling GitHub Pages, the documentation is available at:

https://<github-username>.github.io/pindocs/wbs.html

## Updating the Docs

Edit `data.yaml` and commit the changes.
The website updates automatically.
